Shell 常用命令学习

发表于 2019-01-04 | 分类于 Research | | 阅读次数 | 阅读量 次

Git Bash 中可执行Shell 脚本的命令 Bash 类似编译器作用 全称 ‘Bourne-Again SHell’
Shell 可以看成是编程语言。这里是对个人使用的一些命令及相应语句的记录。

ls -la 显示当前文件夹的内容 或者 ls -la [文件名] 显示指定文件信息
echo 命令用于向窗口输出文本
cat [文件名] 显示文件中的内容
chmod +x ./[文件名] 使得./[文件名]具有 执行权限
[命令or脚本or可执行程序]<< tag 将脚本内开始标记tag 和结束标记tag 之间的内容作为输入
PS: < tag 之间 不留空格会不能正常在网页中显示。
./abc.exe 执行 abc.exe 可执行程序
sed命令
sed -i “1c hi” text.dat # 将文件名为test.dat的第一行替代为hi, 多行如何替换？
我的第一个循环脚本

touch “文件名” #创建文件
rm -if “文件名” #删除文件
command1 | command2 这里的“|” 是把第一个命令的执行结果作为第二个命令的输入
sed -n 23p rw3d.dat | awk -F “ “ ‘{print $3}’ 这里是读取文件rw3d.dat 中的第23行，第三列内容，列以空格分开
awk ‘NR==23 {print $3}’ rw3d.dat 与上一行命令的作用一致
